# Implementation Plan: Modern Retail Management System (POS)

Build a high-performance, dark-themed Retail ERP using Flask, Vanilla JS, and Supabase.

## 🏗️ System Architecture

### Backend (Python/Flask)
- **Framework**: Flask (REST API)
- **Authentication**: Supabase Auth (JWT handling)
- **Database**: Supabase PostgreSQL
- **Security**: Flask-CORS, JWT validation, Input sanitization

### Frontend (Vanilla JS/CSS/HTML)
- **Design**: Modern Dark UI (Indigo/Navy/Green)
- **State Management**: Local Vanilla JS (No heavy frameworks for speed)
- **Icons**: Lucide or FontAwesome
- **Charts**: Chart.js for dashboard analytics

## 🗄️ Database Schema (Supabase)

```sql
-- Profiles: Linked to auth.users
CREATE TABLE profiles (
    id UUID REFERENCES auth.users ON DELETE CASCADE PRIMARY KEY,
    email TEXT UNIQUE,
    role TEXT DEFAULT 'staff', -- admin, staff
    full_name TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Categories
CREATE TABLE categories (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name TEXT UNIQUE NOT NULL
);

-- Products
CREATE TABLE products (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name TEXT NOT NULL,
    sku TEXT UNIQUE NOT NULL,
    category_id BIGINT REFERENCES categories(id),
    price DECIMAL(10,2) NOT NULL,
    stock INT DEFAULT 0,
    low_stock_threshold INT DEFAULT 10,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Customers
CREATE TABLE customers (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name TEXT NOT NULL,
    phone TEXT UNIQUE NOT NULL,
    email TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Sales
CREATE TABLE sales (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    total_amount DECIMAL(10,2) NOT NULL,
    gst_amount DECIMAL(10,2) NOT NULL,
    grand_total DECIMAL(10,2) NOT NULL,
    customer_id BIGINT REFERENCES customers(id),
    user_id UUID REFERENCES profiles(id),
    sale_date TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Sale Items
CREATE TABLE sale_items (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    sale_id BIGINT REFERENCES sales(id) ON DELETE CASCADE,
    product_id BIGINT REFERENCES products(id),
    quantity INT NOT NULL,
    unit_price DECIMAL(10,2) NOT NULL
);

-- Purchases (Procurement)
CREATE TABLE purchases (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    supplier_name TEXT NOT NULL,
    total_cost DECIMAL(10,2) NOT NULL,
    purchase_date TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    user_id UUID REFERENCES profiles(id)
);

-- Purchase Items
CREATE TABLE purchase_items (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    purchase_id BIGINT REFERENCES purchases(id) ON DELETE CASCADE,
    product_id BIGINT REFERENCES products(id),
    quantity INT NOT NULL,
    unit_cost DECIMAL(10,2) NOT NULL
);

-- Business Settings
CREATE TABLE settings (
    id INT PRIMARY KEY DEFAULT 1,
    business_name TEXT DEFAULT 'My Store',
    gst_percent INT DEFAULT 18,
    address TEXT,
    phone TEXT,
    invoice_footer TEXT,
    CONSTRAINT single_row CHECK (id = 1)
);
```
